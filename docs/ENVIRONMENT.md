环境配置

- 使用 `uv 0.8.13` 在项目根目录创建专用虚拟环境：`uv venv .venv`
- 激活环境：`source .venv/bin/activate`
- 初始化项目：新增 `pyproject.toml`
- 依赖安装（使用 `uv add` 写入并安装）：
  - `torch==2.9.1`
  - `torchvision==0.24.1`
  - `numpy==2.3.5`
  - `pandas==2.3.3`
  - `matplotlib==3.10.8`
  - `scikit-learn==1.8.0`

Python 版本：`Python 3.13.7`

虚拟环境位置：项目根目录下的 `.venv`
